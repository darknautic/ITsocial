CSS Reference Checklist and Cheatsheet

CSS (Cascading Style Sheets) adds the styling to your web pages, including creating the page's layout, navigation bar, colors, fonts, table and form styling, etc. It goes hand in hand with the HTML. There are a number of outstanding free and inexpensive online resources for learning CCS including Codecademy.com (free), Lynda.com (subscription) and TeamTreehouse.com (subscription). They are all good, and are interactive so you reinforce what you are reading. Like HTML, CSS is not difficult. You just have to learn the main concepts and practice it. So take one of these courses. After you complete the course you can use the below checklist/cheat sheet to ensure you have all the concepts down solid and use it as an ongoing reference.

So, what do I do with this checklist/cheat sheet?

  • It is meant to be used after you take a CSS course. It doesn't teach new concepts, rather it reinforces the concepts you learned in a course.
  • Copy and paste it to your PC and use it as a reference. Put it in Excel, or Word or PDF or whatever format you like for editable reference material.
  • Add to it or modify it as you see fit.
  • Print it out and make sure you understand every concept in it. Go back to the course if you aren't clear about a concept, or use other resources. W3Schools.com is a great free reference site for all these concepts.

Note: this CSS Checklist/Cheat Sheet does not reference Bootstrap classes. Use it to understand the Bootstrap stylesheet classes and to customize your site's style.

CSS Checklist/Cheat Sheet

Where to put CSS styling (3 methods):

  1. External Style Sheet: This is the recommended method.
    Create a separate file. It can be named anything but must end in .css.
    Best practice to put it in a separate css directory.
    Add a link to the stylesheet in the head section of the HTML page <link rel="stylesheet" type="text/css" href="css/style.css">
  2. Internal Style Sheet in the head section of the HTML page.
    This is not recommended in the final product, but useful when developing web pages.
    In the head section place within <style> tags <style type="text/css">Add your CSS here</style>
  3. In-line Style
    Add style directly in the HTML opening tags of an element as a style attribute
    Uses HTML syntax rather than CSS syntax.
    Generally avoid using it, but can be used to style a single unique element.
    sytnax: <tag style="attribute: value">
    example: <h1 style="text-align: center">
    Use semicolon with multiple attributes <h1 style="text-align:center; color:red">

If there are multiple styles specified, the order of priority is:

  1. Inline style - inside an HTML element (highest priority).
  2. Internal style sheet - in the head section
  3. External style sheet - this is the recommended location. If you link to multiple stylesheets, the last one has precedence.
  4. Browser default (lowest priority)

But If the link to the external style sheet is placed after the internal style sheet in the head section, then the external style sheet will override the internal style sheet!

Terminology:

  • Element: In an HTML document it is everything between a start and end tag (e.g., body, h1, table, div, form, p, etc.)
  • Style rule: sets the style for a specified HTML element. h1 {color:blue; font-size:14px;}
  • A style rule has two main parts: a selector, and one or more declarations: selector {declaration;}
  • Selector: the HTML element you want to style. h1 is the selector in the example
  • Declaration: specifies how content described by the markup looks. {color:blue; font-size:14px;} are the declarations
    A declaration consists of two subitems separated by a colon:
    • Property: the style attribute you want to change. Style attributes affect how the user's computer displays text and graphics. In the example, color and font-size are the properties.
    • Value: is what you want the style attribute to be. In the example blue and 14px are the values
  • Id: individual elements can be given an id. You can set style specifically for that id. Uses #selector
  • Class: multiple elements can be given a class. You can set the style for a class or elements. Uses .selector

CSS Syntax:

  • CSS Syntax applies to external and Internal Stylesheets. Inline style uses HTML syntax.
  • Comments: /* Comments */
  • Sytnax example: h1 {color:blue; font-size:14px;}
  • Curly braces { } enclose the CSS declarations.
  • A semicolon always follows a declaration so the browser can distinguish them. Putting a semicolon after the last declaration is optional but good to include to be consistent.
  • A colon separates the property and value.
  • To set style rules for the entire document, set them in the body element. body {background-color:grey;}
  • Group selectors: To have multiple selectors use the same style, list them separated by commas in front of the declaration. h1, h2, h3 {color: blue;}
  • Nest selectors: you can style a selector for a specified class. In this case paragraphs with <p class="blue"> - .blue p {color:blue}
  • You can set style rules for sections of a document with the div or span elements, or the HTML5 header, nav, aside, and footer elements.

Tips and tricks

  • To eliminate the extra line between <p> paragraphs style the margin at 0. p{ margin:0; }
  • Put two different styles on the same line (e.g., header and paragraph text) <h1 style="display: inline;">Header text</h1> <p style="display: inline;">regular text</p>
  • To reduce the space after <h1>, <h2> etc: h1{ margin-bottom:0px;} or p{margin-top: 0px;}
  • Create a blank line use <P> without the closing </p>, or <br/>

Colors

  • Colors are specified by: a Hexadecimal (or HEX) value like #ff0000, an RGB (red green blue) value like rgb(255,0,0), or a color name like red
  • Color Listings: www.w3schools.com/html/html_colors.asp
  • HEX values like #000000 are actually three pairs of two digit values representing red, green and blue. You can use three digit hex numbers as shortcuts such as #000 instead of #000000.
    Hex values: www.w3schools.com/html/html_colorvalues.asp
  • common colors: black #000000; gray #808080; maroon #800000; purple #800080; green #008000; olive #808000; navy #000080; teal #008080; silver #C0C0C0; white #FFFFFF; red #FF0000; fuchsia #FF00FF; lime #00FF00; yellow #FFFF00; blue #0000FF; Aqua #00FFFF
  • Color names: www.w3schools.com/html/html_colornames.asp
  • Color Mixer: www.w3schools.com/tags/ref_colormixer.asp
  • You can determine the color hex value in an image with Adobe Photoshop using the eyedropper tool.

Styling Backgrounds

  • Put the background styling in the body selector. body {background-color:#f9f7f5;}
  • You can specify the color as a hex value #f9f7f5, an rgb value rgb(249,247,245), or a name value GhostWhite
  • Add a background image. By default, the image is repeated so it covers the entire element. body {background-image:url('wallpaper.png');}
  • Background-Repeat horizontal: To repeat a background image only horizontally use background-repeat:repeat-x. body {background-image:url('wallpaper.png'); background-repeat:repeat-x;}
  • Background-Repeat vertical: To repeat a background image only vertically use background-repeat:repeat-y. To repeat only vertically use y instead of x. body {background-image:url('wallpaper.png'); background-repeat:repeat-y;}
  • Background-No-Repeat: To not repeat a background image use background-repeat:no-repeat; body {background-image:url('wallpaper.png'); background-repeat:repeat-no-repeat;}
  • To fix a background image so it doesn't scroll use background-attachment:fixed. body {background-image:url('wallpaper.png'); background-attachment:fixed;}
  • Background position: Default position of a background image is top left. To change that use the background-position property. body {background-image:url('wallpaper.png'); background-position:right top;}
    The value has two parts, first left, center, or right, then top, center, or bottom.
    You can set it by x% y% with % relative to the page.
    or by xpos ypos.
  • The shorthand background property sets all background properties. body {background:#f9f7f5 url('wallpaper.png') no-repeat right top;}
    The order is background-color, background-image, background-repeat, background-attachment, background-position. You can leave out one or more properties.
  • Change the opacity of a background image by using the opacity parameter. Note for IE8 and earlier you need to use a the filter:alpha parameter. Best to specify both. The example makes the image 50% transparent. img{opacity:0.5; filter:alpha(opacity=50);}

Measurement Units:

CSS properties that require size values can be given in the below measurement units varying by property.
The below are absolute lengths except % which is relative.
  • em - 1em is equal to the current font size. 2em means 2 times the size of the current font.
  • % - percentage
  • in - inch
  • cm - centimeter
  • mm - millimeter
  • ex - one ex is the x-height of a font (x-height is usually about half the font-size)
  • pt - point (1 pt is the same as 1/72 inch)
  • pc - pica (1 pc is the same as 12 points)
  • px - pixels (a dot on the computer screen). Default font-size on browsers is 16px, so 1em=16px.

Styling Text

  • Color: Set text color using the color property. You can use hex, rgb or name values body {color:blue;} or h1 {color:#00ff00;} or h2 {color:rgb(255,0,0);}
  • Text-align: Default is left, but can be changed to center, right or justify (spread evenly across the page): h1 {text-align:center;} or h1 {text-align:right;} or h1 {text-align:justify;}
  • Remove underline from hyperlinks using text-decoration:none on the anchor tag. a {text-decoration:none;}
  • Use text-decoration for underline, line-through, or overline text h1 {text-decoration:underline;} or h1 {text-decoration:line-through;} or h1 {text-decoration:overline;}
  • Text-indent: indent the first line of a paragraph p {text-indent:50px;}
  • Line-height: set the line height within a paragraph. Default in most browsers is 110 to 120%. body{line-height:100%;}
  • Double-space your textbody{line-height:200%;}
  • Vertical-align: Align images within a line of text. Default is to align at the bottom of the line. You can change it to middle or top: img {vertical-align:text-top;}

Styling Font

CSS font properties define the font family, size, boldness and style of text.

font-family
  • List of commonly used font combinations: www.w3schools.com/cssref/css_websafe_fonts.asp
  • There are two types of font family names: generic family (e.g., sans-serif) and font family (a specific font family like Arial)
  • Generic font families are font families with a similar look such as serif, sans-serif, comic, and monospace.
  • Difference Between Serif and Sans-serif Fonts - Serif font has tails on the ends of letters. San-serif does not. On computer screens, sans-serif fonts are considered easier to read than serif fonts.
  • Monospace: All monospace characters have the same width.
  • The font-family property should hold several font names as a "fallback" system. If the browser does not support the first font, it tries the next font.
  • Start with the font you want, and end with a generic family, to let the browser pick a similar font in the generic family, if no other fonts are available.
  • Separate the fonts with a comma.
  • Put quotes around font families that are more than one word p{font-family:"Times New Roman", Times, serif;}
  • Sans-serif font family example p{font-family:Verdana, Geneva, sans-serif}
Font-size
  • The default font size is 16 pixels for regular text
  • You can set the font size in absolute terms by specifying the size in pixels (px), points (pt), or inches (in). h1 {font-size:40px;} or h1{font-size:20pt;} or h1{font-size:.3in;}
  • Or set it in relative terms using em or %. 1em equals the browser's default font size so 1em for regular text is 16 pixels. 1em is the same as 100%. 1.2em is the same as 120%. h1 {font-size:1.2em;} or h1{font-size:120%;}
  • Using relative size em makes a site more responsive in adapting between computer screens and mobile devices and is therefore the recommended measuring unit. p{font-size:1.1em;}
  • Formula to convert from pixel to em: px/16=em
Font-style
  • Font-style: Make italic {font-style:italic;}
  • Font-variant: convert text to small caps {font-variant:small-caps;}
  • Font-weight: make the text bold, bolder, lighter, or specify using 100 to 900. {font-weight:bold;} or {font-weight:lighter;} or {font-weight:500;}
Font shorthand
Font: shorthand to set all font properties in one declaration. Set each Font property in this order: font-style, font-variant, font-weight, font-size/line height, font-family. You don't have to include properties that will use the default. The line-height property sets the space between lines. {font:italic bold 1.1em Georgia,serif;} or {font: 12px/10px Arial;}.

IDs

  • Use IDs when the style will be unique to one element in the HTML document.
  • First create the ID in the opening HTML tag: <tag id="id_name">
    Example: <div id="container1"> or <p id="paragraph1">
  • Then in the CSS stylesheet, create the style rule with a hash sign # followed by the id name as the selector. #selector{parameter:value; parameter:value;}
    Example: #paragraph1{text-align:center; color:blue;}

Classes

  • Use classes when the same style will be applied to multiple elements.
  • First create the class in the opening HTML tag: <tag class="class_name">
    Example: <form class="standard_form">
  • Then in the CSS stylesheet, create the style rule with a period . followed by the class name as the selector. .selector{parameter:value; parameter:value;}
    Example: .standard_form{margin: 10px 20px; padding: 15px 20px;}
  • To assign more than one class to an opening tag, separate the classes with a space. <div class="container maincontent">

Styling Links and pseudo classes

Pseudo Classes
  • Pseudo-classes add special effects to some selectors.
  • Pseudo class syntax (Pseudo class names are not case sensitive) selector:pseudo-class {property:value;}
  • Focus: used in forms to change an input box when it's clicked on. input:focus { background-color:aquamarine; }
Links with Pseudo Classes

There are four states of a link. You can change their color, background-color, or remove the underline. With this pseudo class the selector is the anchor tag a

  1. a:link - normal, unvisited link. Ex: a:link {color:blue;} or a:link {text-decoration:none;} or a:link {background-color:transparent;}
  2. a:visited - link after the user has visited. Ex: a:visited {color:darkblue;} or a:visited {text-decoration:none;} or a:visited {background-color:transparent;}
  3. a:hover - a link when the user's mouse is hovering over it. Ex: a:hover {color:white;} or a:hover {text-decoration:underline;} or a:hover {background-color:darkblue;}
  4. a:active - a link the moment it is clicked. Ex: a:active {color:azure;} or a:active {text-decoration:underline;} or a:active {background-color:blue;}

Order rules: To work in all browsers, put a:active after a:hover, and a:hover after a:link and a:visited.

Styling Lists

  • Default marker for unordered list is a bullet point. Default for an ordered list is a number.
  • list-style-type: Changes the list marker
  • Options for unordered list besides the default disc are none, circle, or square. Ex: ul {list-style-type: none;} or ul {list-style-type: circle;}
  • Options for ordered list besides the default decimal are lower-alpha, upper-alpha, lower-roman, upper-roman. Ex: ol {list-style-type: lower-alpha;} or ol {list-style-type: upper-roman;}
  • To only change some lists, add an id or class to the opening tag of the list. Ex: <ul id="list1"> or <ol class="listtype1">
  • Then in the CSS stylesheet set the list id or class style. Ex: ul#list1 {list-style-type: square;} or ol.listtype1 {list-style-type: lower-alpha;}

Styling Tables

  • You can style one or more of the different table elements including table, caption, thead (table header), th (column header), tbody, tr (row), td (cell), tfooter (table footer).
  • Set the style for all table elements or use ids and classes to style individual tables or classes of tables.
  • As default there are spaces between the table cell borders. To make the borders into a solid line use: table { border-collapse:collapse; }
  • Set borders. Ex: table, th, td { border: 1px solid black; }
  • Table Width and Height properties. Ex: table { width:100%; } th { height:20px; }
  • Text-align property sets the horizontal alignment, like left, right, or center. Ex: td { text-align:right; }
  • Vertical-align property sets the vertical alignment, like top, bottom, or middle. Ex: td { height:20px; vertical-align:bottom; }
  • Set padding. Ex: td { padding:10px; } or td, th { padding:10px; }
  • Change color of borders, background and text. Ex: table, td, th { border:1px solid gray; } or th { background-color:gray; color:white; }
  • To alternate the color of table rows add a class attribute to the opening tag of every other row. Ex: <tr class="colorrow">
  • Then in the CSS file create a background color for the class. Ex: tr.colorrow td {background-color:LightGray}
  • Change the location of the table caption from the default of top to the bottom. caption {caption-side:bottom;}

CSS Box Model

  • Consists of: margins, borders, padding, and content.
  • Content is on the inside, surrounded by the padding, then the border, then the margin. Padding, border and margin can be set to zero/none.
  • Total width of an element = width + left padding + right padding + left border + right border + left margin + right margin
  • Total height of an element =height + top padding + bottom padding + top border + bottom border + top margin + bottom margin
  • For margins and padding use px pixels rather than pt points or em because the increments are generally smaller.
Shorthand for styling individual sides of a box:
  • To style the border, margin or padding of the sides of a box differently you can use shorthand.
  • List the sides as multiple values to a parameter separated by spaces.
  • The order is Top, Right, Bottom, Left (clockwise).
  • To give each side a different style list values for all four sides. Ex: {border-style:solid dashed double dotted;}
  • To give the top and bottom sides one style, and the right and left sides another style, list two values to the parameter. The first parameter is for the top and bottom, the second is for the right and left. Ex: {border-style:dashed solid;}
  • To set the top one style, the bottom another style, and right/left a third style, list three styles. The second is for both right and left. Ex: {border-style:solid dashed double;}
Styling Borders
  • Borders can be used around tables, forms, images, paragraphs, etc.
  • Borders come between the margin and padding.
  • The default border-style is none. To show a border you must give the border-style a value.
  • border-style: options are solid (default), none, hidden, dotted, dashed, double, groove, ridge, inset, outset. Ex: table{border-style:inset;} or table{border-style:dotted;}
  • border-width: default is medium (3px), other options: thin (1px), thick (5px). Ex: table{border-width:2px;} or table{border-width:thick;}
  • border-color: use hex number, color name, or rgb number. You can also set it to transparent. Ex: table{border-color:#808080;} or table{border-color:gray}
  • border: Or use the shorthand border property to set all three in the order of width, style then color. Listing border style is required, the other two are not. Ex: table{border: 2px inset gray}
  • You can set one side using border-top, right, bottom, or left. Ex: {border-top-style:none} or {border-right-width:5px} or {border-bottom: 3px double gray}
  • To give the top and bottom one style and right and left another style using shorthand. Ex: {border-style:dashed solid;}
  • To give the top one style, bottom another style, and right/left a third style. Ex: {border-style:solid dashed double;}
  • To give each side a different style. Ex: {border-style:solid dashed double dotted;}
  • Create a border around a paragraph by assigning the paragraph tag to a class (or id if just one). Ex: <p class="p_border">
    and in the CSS stylesheet give the class a border. Ex: .p_border {border: 3px solid gray;}
Styling Outlines
  • Borders go between the padding and the margin, but outlines go outside the margin and are not part of an element's dimensions.
  • Styling options, rules and syntax are the same as for borders. Just replace border with outline. Ex: table{border: 2px inset gray}
Styling Margins
  • Margin clears an area outside the border of an element. By default the margin has no background color.
  • The default margin value is auto which lets the browser set the margin.
  • Set margins to absolute widths (e.g., px) or set them as a percent of the containing element with %. Ex: table{margin:10px;} or table{margin:10%;}
  • The margin parameter sets all sides the same width.
  • Set individual sides with margin-top, right, bottom, or left. Ex: {margin-top: 5px; margin-bottom:10px}
  • You can overlap content with negative margins.
  • Use margin shorthand to set multiple sides at once
    Set all four sides to different widths using shorthand (set clockwise). Ex: {margin: 5px 5px 10px 15px;}
    If you use three values, the second value sets the right and left sides. Ex: {margin: 5px 15px 10px;}
    If you use two values the first sets the top and bottom and the second sets right and left. Ex: {margin: 5px 10px;}
Styling Padding
  • Padding clears space between the element's content and the border. The padding is affected by the element's background color.
  • Set padding to absolute widths (e.g., px) or set it as a percent of the containing element with %. Ex: table{padding:10px;} or table{padding-top:10%;}
  • Styling options, rules and syntax are the same as for margins. Just replace margin with padding.
    Ex: {padding-top: 5px; padding-bottom:10px}

Set dimensions - width, height

  • You can set specific height and width of an element using length or %. Ex: {width:20px;} or {width:50%}) or {height:100px}
  • You can set a maximum or minimum width or height. Ex: {max-width:400px} or {min-height:500px}

Images

  • You can set the dimensions of an image directly in the image tag. Ex: <img src="imagefile.jpg" width="500" height="100"/>
  • Or give the image an id or class <img src="imagefile.jpg" id="image1"/> or <img src="imagefile.jpg" class="wideimage"/>
    Then set the dimensions in the CSS file. Ex: #image1{width:500; height:100;} or .wideimage{width:500; height:100;}
  • If you don't specify the image dimentions it will default to auto and use the actual dimensions of the image file.
  • You can specify only width or only height and the image will stretch or shrink proportionally.
Image Opacity / Transparency
  • You can make an image partially transparent. This is often used for background images that have text on top of them.
  • The CSS3 syntax uses (opacity:x) where x is 0.0 to 1.0. IE8 and older use filter:alpha(opacity=yy) where yy is 1 to 100. The higher number mean less transparency.
  • To cover the majority of browsers use both methods.
    Example with image 50% transparent.img{opacity:0.5; filter:alpha(opacity=50);}

Positioning

  • Set the position of an element using the position property (static, fixed, relative, absolute) and setting the top or bottom, right or left properties.
  • You can specify absolute lengths (standard is px) or as %, or leave it unspecified which defaults to auto.
  • Static: is the default position property and not used with the top/bottom, right/left properties.
  • Fixed: you can fix an element, like a background image or the top navigation bar, so that it does not scroll. Other elements ignore fixed elements and overlap them. Ex: {position: fixed;top: 50px; right:20px; }
  • Relative: you can move an element from where it would normally be using relative positioning. This allows you to overlap other elements. The normal space is still reserved for the element. Ex: { position:relative; left:-10px; } or { position:relative; left:10px; } or { position:relative; top:-30px; }
  • Absolute: like fixed except it will scroll off the screen. Ex: h1{ position:absolute; left:20px; top:30px; }
Overlapping Elements
  • If using the fixed, relative, or absolute positioning causes elements to overlap, you can use the z-index property to specify which is on top (higher number is on top). Ex: {position:absolute; left:0px; top:0px; z-index:1}
  • If elements overlap and a z-index is not specified, the last element in the HTML code goes on top.
  • 1 or higher will go on top of an element without a z-index specified, and -1 or lower will go behind it.
  • You can use it to put text over an image.

Change the Cursor

  • Change the cursor when the mouse is over an element using the span element and the cursor property. Ex: <span style="cursor:wait">Put the text, image, etc here.</span>
  • Cursor styles include crosshair, pointer (hand), help (question mark), move, wait (hourglass), text, ne-resize, ne-resize, nw-resize, n-resize, s-resize, se-resize, sw-resize, w-resize, auto, default.

Overflow

  • If text overflows an element like a box you can set it to scroll using the auto or scroll properties (auto adds a scroll bar only if needed). .textbox{width:300px; height:300px; overflow:auto;} or #textbox1{width:300px; height:300px; overflow:scroll;}
  • You can set it to not show overflow text using hidden, or to show it beyond the box using visible. Visible is the default for overflow. Ex: .textbox{width:300px; height:e00px; overflow:hidden;} or #textbox2{width:300px; height:300px; overflow:visible;}

Float

  • Commonly used with images and layouts. Specifies if an element should float horizontally (to the left or right), letting other elements wrap around it. Not used vertically. img {float: right;} or img{float: left;}
  • To prevent elements from flowing around an element to the right, left or both sides use the clear property. {clear:left;} or {clear:right;} or {clear:both;}

Horizontal Alignment

  • Use horizontal alignment with block elements to create your page layout.
  • Block elements take up the full width available and have a line break before and after them. Examples of block elements are <h1>, <p>, <div>, <li>
  • Inline elements like <img>, <a>, and <span> only take up as much room as needed, not the whole line.
  • It is common to create a div with a container class for your main content area. <div class="container">
  • And set the width for the class in the CSS stylesheet. .container {width: 940px;}
  • To avoid issues with browsers it's a good idea to set the default margin in the body to 0. body{margin:0px;}
  • To center an element like a container, use the auto margin to make it equal on both sides. .container {margin-left:auto; margin-right:auto;} or just .container {margin:auto;}

Navigation Bar

  • Create a horizontal nav bar by creating a list of hyperlinks in an unordered list put inline, styled to remove the underlines and bullet points.
  • Create a div for the nav bar or use the HTML5 nav tags. Assign it to one or more classes to do the styling. <div class="navbar">put your nav bar stuff here</div>

Mandatory styling includes:

  • Set the navbar's background color and height. .navbar {min-height: 40px; background: #black;}
  • Display the navbar links as an inline block .navbar li {display: inline-block;}
  • Remove the underline from the links and change the text color. .navbar li a {text-decoration: none; color:white}
  • Remove the bullets from the unordered list. .navbar {list-style-type: none;}
  • You will also need to set the appropriate margin and padding lengths and may need to adjust the relative positioning.

Attribute Selectors

  • Used to style HTML elements that have specific attributes, not just class and id. Often used with forms.
  • use square brackets [] around the attribute. Ex: input[type="submit"] {cursor: pointer;}

Misc

  • You can make an element invisible but still have it take up the space. .hide {visibility:hidden;}
  • Or you can make the element disappear and not take up the space .hide {display:none;}
  • Use <center> </center> tags to have everything between them be centered. Like for an image in a carousel.
